Detailed Description

This class is used to facilitate the paints of different graphs.

This class defines some usefull procedures for some basic standard drawings, all the functios in this class are static meanings that can be called without the need to make an instance of the class itself

This procedure draws a single bar graph.

Parameters
qp: the painter you want to use for the drawing
startPos: the start position of the rectangle (left top corner)
width: the width of the rectangle
height: the height of the rectangle
value: the value in percentage of the width for the lenght of the bar
drawText: Whether or not to draw the text percentage
color1: The starting color of the bar (color gets interpolated with color2)
color2: The ending color of the bar (color gets interpolated with color1)
