Detailed Description

This class is an abstract class used to generate classes for data fetching.

This class is is used as a base for implementing different type of classes used to query different type of data. You will need to reimplement the virtual fucntions getData and destructor The data you will fetch will need to be stored in the m_cache member which is a 2D std::vector<float> matrix. You also have a couplw of variables you can use in your computation if needed :

  • m_index
  • m_multi
