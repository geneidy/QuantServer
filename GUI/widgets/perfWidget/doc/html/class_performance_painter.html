Detailed Description

This is an abstract painter.

This class define an abstract painter that can be subclassed to generate different type of paintings for the performance widget

In order to provide a little bit of flexibility this class implements a couple of generic attriutes and relative setter functions , this attributes can be used or not from the subclasses to try to give a bit of customizzation Those attributes are :

  • m_color1
  • m_color2
